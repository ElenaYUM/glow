Как получить токен

  1. Откройте github.com → ваш профиль → Settings.
  2. Внизу слева: Developer settings → Personal access tokens → Fine-grained tokens → Generate new token.
  3. Repository access → Only select repositories → выберите репозиторий магазина (glow).
  4. Permissions → Repository permissions → Contents → выберите Read and write.
  5. Нажмите Generate token, скопируйте строку токена и вставьте её в поле слева.
  6. Нажмите «Сохранить и проверить» — должно появиться «Подключено ✓».

Токен — это как пароль для записи в репозиторий. Никому его не показывайте. Если потеряли — просто создайте новый, старый удалите на GitHub.

Первый раз нажмите «⬆️ Выгрузить ВСЕ данные в GitHub» — это создаст файлы данных в репозитории. Дальше всё сохраняется автоматически при каждом изменении.
