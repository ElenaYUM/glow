# 🛍️ GLOW — интернет-магазин корейской косметики

Полнофункциональный магазин на чистом HTML / CSS / JavaScript в стиле
современного e-commerce (референс — tiki-toki.by): белый фон, красный акцент,
поиск, категории, рейтинги, акции. Данные хранятся в браузере (`localStorage`),
сервер не нужен.

## 🚀 Запуск
1. Откройте папку `GLOW` на рабочем столе.
2. Дважды кликните по **`index.html`**.

## 🛍️ Для покупателей
- Гео-слайдер на главной, сетка из 6 категорий
- Блоки **Акции**, **Новинки**, полный **Каталог** с фильтрами
- Поиск по названию и бренду
- Карточки товаров: рейтинг ⭐, отзывы, скидка −%, старая зачёркнутая цена,
  кнопки «В корзину» и «Быстрый заказ»
- Корзина: изменение количества, удаление, подсчёт выгоды и итога, оформление
- Бренды, отзывы, блог, рассылка, адаптив под телефон

## 🔐 Админка
Вход: кнопка **«Админ»** в шапке или файл **`admin.html`**.
**Пароль:** `glow2026`

Возможности: статистика, добавление / редактирование / удаление товаров,
поля (название, бренд, категория, цена, **старая цена** для скидки, остаток,
**рейтинг**, **кол-во отзывов**, описание, бейдж ХИТ/NEW, иконка, цвет упаковки),
поиск, сброс каталога. Всё сразу отражается в магазине.

## 🔑 Смена пароля
В `js/store.js`: `const ADMIN_PASSWORD = "glow2026";`

## 📁 Структура
```
GLOW/
├── index.html        # витрина
├── admin.html        # админ-панель
├── css/  styles.css, admin.css
└── js/   store.js (данные), shop.js (витрина), admin.js (админка)
```

## ⚠️ Важно
- Каталог и корзина хранятся в `localStorage` браузера.
- Пароль проверяется на стороне браузера — это демо-защита, не для боевого магазина.
- Картинки товаров — эмодзи на нейтральном фоне (чтобы работало офлайн без файлов).
  Можно заменить на реальные фотографии.
